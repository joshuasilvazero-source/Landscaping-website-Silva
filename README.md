# EverGreen Landscaping Pro

A premium responsive landscaping website portfolio project built with HTML, CSS, and JavaScript.

## Features

- Sticky glassmorphism navigation bar
- Scroll shadow effect
- Mobile menu
- Dark mode toggle
- Professional hero section
- Services section
- Process section
- Project cards with images
- Booking form demo
- LocalStorage form saving
- Scroll reveal animations
- Fully responsive layout

## Tech Stack

- HTML
- CSS
- JavaScript
