# EverGreen Landscaping Website

A modern, responsive landscaping business website built with HTML, CSS, and JavaScript.

## Features

- Premium hero section
- Responsive mobile navigation
- Dark mode toggle
- Service cards
- Project highlights
- Booking form demo
- LocalStorage booking save
- Clean code structure

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Purpose

This project was built as a professional front-end portfolio project to demonstrate responsive design, UI/UX layout, JavaScript interaction, and real-world business website structure.
